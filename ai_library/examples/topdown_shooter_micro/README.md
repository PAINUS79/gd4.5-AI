# Top-Down Shooter Micro Example

## Purpose
Minimal working top-down shooter demonstrating:
- TopDownMove2D usage
- Projectile system
- Enemy chase AI
- Combat mechanics

## How to Use
1. Load the main scene
2. Use WASD or arrow keys for movement
3. Use mouse/action button to shoot
4. Defeat enemies, survive waves
